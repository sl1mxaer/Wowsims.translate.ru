# Wowsims.translate.ru

Браузерное расширение для перевода tooltips на сайте https://www.wowsims.com на русский язык через ru.wowhead.com.

## Установка (для разработки)
1. Скачай репозиторий
2. Открой в браузере `chrome://extensions/`
3. Включи "Режим разработчика"
4. "Загрузить распакованное расширение" → выбери папку с файлами

## Как работает
- Автоматически заменяет ссылки wowhead.com → ru.wowhead.com
- Работает с Gem Summary, Gear, Reforge Summary
- Поддерживает динамические обновления страницы
